# Jäger kleiner Beute

**Regel:** Erhält +2 auf alle Würfe (Angriff, Verteidigung, Ausweichen) gegen Ziele mit einer Basis-Gesundheit von 3 oder weniger.

## Karten mit dieser Fähigkeit

```dataview
TABLE WITHOUT ID   
	file.link as "Tier",   
	sterne as "Sterne" 
FROM #tierkarte
WHERE contains(file.outlinks, this.file.link)
````
