# Sozialer Schutz

**Regel:** In einer Gruppe der Größe 3+ kann es nicht Ziel von gegnerischen
[[Spezialaktionen]] sein.

## Karten mit dieser Fähigkeit

```dataview
TABLE WITHOUT ID   
	file.link as "Tier"
FROM #tierkarte
WHERE contains(file.outlinks, this.file.link)
````
